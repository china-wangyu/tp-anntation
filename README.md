# tp-anntation



## 简介

ThinkPHP5 的简易注解扩展，含注解路由（支持路由分组），注解参数验证（支持验证器和参数验证），注解Api文档生成等功能


## 安装

```bash
composer require wangyu/tp-anntation
```
